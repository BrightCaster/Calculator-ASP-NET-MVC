# Calculator-ASP-NET-MVC
The usual calculator is based on ASP NET MVC

<p>You can use the calculator:</p>
<ul>
<li>
<p>as a normal keyboard input</p></li>
<li>
<p>by pressing the built-in buttons</p></li>
</ul>
